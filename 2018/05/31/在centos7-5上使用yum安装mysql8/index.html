在centos7.5上使用yum安装mysql8

发表于 2018-05-31
修改mysql的密码策略做了很好的总结，但，注意一点，在mysql8.0.1中密码策略变量的变化；由“_”连接符改变为”.”连接符；

查看password密码策略的变量名 SHOW VARIABLES LIKE ‘validate_password%’;

set global validate_password.policy=0;

set global validate_password.length=4;
